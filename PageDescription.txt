Интерфейс

Веб-приложение имеет четыре модуля: 

Режим приложения

Верхняя панель, включающая следующие элементы:
•	Селектор управление версиями, содержащий значения V1/V2/V3
•	Чекбокс управление ошибками

Авторизация

Страница входа, позволяет авторизовать существующего клиента (переход на страницу Главная) или перейти к странице Регистрация. Включает следующие элементы:
V1:
•	Текст название приложения
•	Поле логин (допустимы любые значения)
•	Поле пароль (допустимы любые значения)
•	Чекбокс запомнить меня
•	Кнопка войти
•	Кнопка зарегестрироваться
•	Кнопка восстановить доступ
V2:
Помимо элементов из V1 присутствуют 
•	Поле идентификатор сессии (допустимы любые значения)
•	Чекбокс – войти по идентификатору сессии
V3:
	Аналогичен V2, но использует медленную прогрузку контента

Регистрация 

Страница, обеспечивающая возможность зарегистрировать нового клиента. Содержит следующие элементы:

V1:
•	Поле идентификатор пользователя (допустимы любые значения)
•	Поле имя (допустимы любые значения)
•	Поле страна (Индия, Боливия, Австралия, Ангола, Уругвай, Филиппины)
•	Поле пароль (допустимы любые значения)
•	Кнопка зарегистрироваться
V2:
Помимо элементов из V1 присутствуют 
•	Поле идентификатор сессии (допустимы любые значения)
•	Чекбокс создать идентификатор сесссии автоматически
V3:
	Аналогичен V2, но использует медленную прогрузку контента
  
Главная

•	Лейбл сообщение, содержащее имя клиента и страну
•	Кнопка выйти из системы
